# Sinatra base structure

Just wanted to get a base skelet for my sinatra apps inspired by the rails folder architecture
I've set it up with this concepts:

* mvc architecture
* modular (check config.ru)
* rails assets pipline style
* capybara + minitest


Nothing fancy here :)
